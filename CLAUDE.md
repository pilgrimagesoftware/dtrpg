# DriveThruRPG

Master repository to organize DTRPG repos

## About This Project

This project covers all repos involved in creating a DriveThru RPG SDK and application
for various languages, interfaces, and platforms.

## Submodules

- `dtrpg-sdk` - contains the meta-repository for SDK work
- `dtrpg-app` - contains the meta-repository for Application work
